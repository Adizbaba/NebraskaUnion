# **App Name**: Horizon Financial

## Core Features:

- Homepage Hero Banner: Visually striking hero section with key offers like "Pop Into Perks With a Checking Account! Get a $50 Gift Card + $25 Cash".
- Interest Rate Display: Dynamic display of key interest rates for 6-month share certificates, personal loans, and auto loans, sourced automatically via a rate management tool
- Informational Section: Three-column layout with content for Auto Resource Center, Shared Branching, and Financial Wellness.
- Member Testimonials: Rotating testimonials to build trust, featuring member quotes.
- Footer Navigation: Comprehensive footer with contact details, quick links, and social media integrations.
- Personalized Financial Tips: AI-powered tool that analyzes user's financial behavior and recommends tailored advice to optimize savings and reduce debt. This feature is not explicitly visible on the homepage, but it can guide which interest rates, promotional offers, or resources to highlight for returning users

## Style Guidelines:

- Primary color: Deep blue (#224292) to convey trust and stability.
- Background color: Very light blue (#F0F4FF), almost white, for a clean and modern look.
- Accent color: Warm red (#D9534F) to highlight CTAs and important information, contrasting with the primary blue.
- Body and headline font: 'Inter' for a modern, clean, and readable user experience.
- Clean and modern layout with a focus on readability and easy navigation.
- Subtle hover animations on buttons and links to enhance user engagement.
- Use professional, consistent icons to visually represent different services and sections.
